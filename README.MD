# Cairo Coffee

Landing page responsive para una cafetería ficticia desarrollada con HTML y CSS.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- Responsive Design
- Formspree

## Características

- Diseño responsive
- Navegación con scroll suave
- Menú de productos
- Formulario funcional de pedidos
- Botón para volver al inicio
